##  Temas vistos en clase

### Lenguajes de programación y Python
Un **lenguaje de programación** es un conjunto de reglas y símbolos que usamos para comunicarnos con una computadora y decirle qué hacer.  
Existen muchos lenguajes, como C, Java o JavaScript, y **Python**, que es el que usamos en clase.  
Python es popular porque tiene una sintaxis sencilla, es fácil de leer y se usa en muchos campos como inteligencia artificial, análisis de datos y desarrollo web.

---

### Variables, valores y operadores
Una **variable** es un nombre que se usa para guardar un valor.  
Por ejemplo, podemos guardar la edad de una persona o su nombre.  
Los **valores** pueden ser números, texto o valores lógicos (verdadero o falso).  
Los **operadores** permiten realizar operaciones como sumar, restar o multiplicar.

---

### Orden de las operaciones
En Python, las operaciones se ejecutan siguiendo un orden:  
1. Paréntesis `()`  
2. Potencias `**`  
3. Multiplicación y división `* /`  
4. Suma y resta `+ -`  

Esto significa que, igual que en matemáticas, las multiplicaciones se hacen antes que las sumas.

---

### Operaciones con cadenas de texto
Las **cadenas de texto** (strings) también se pueden combinar o repetir.  
Por ejemplo:  
- `"Hola " + "Mundo"` une dos textos.  
- `"Python " * 3` repite el texto tres veces.  

Esto permite crear mensajes personalizados o trabajar con palabras en lugar de solo números.

---

### Funciones y recursión
Una **función** es un bloque de código que realiza una tarea.  
Se usa para organizar y reutilizar instrucciones.  
Por ejemplo, una función que saluda o que calcula el promedio de notas.  

La **recursión** ocurre cuando una función se llama a sí misma para resolver un problema paso a paso, como al calcular el **factorial** de un número o recorrer una lista.

---
